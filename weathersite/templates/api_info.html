    Daily Weather free API

    Tired of looking for a free API that allows you to know the temperature? You've come to the right place. With our simple functionality you'll have the time of the city you want and even be able to specify the day

    Know our last updated weather on a city
    To know the temperature of a city it is sufficient to make a request to:

    dailyweather.herokuapp.com/api/{cityName}
    or
    dailyweather.herokuapp.com/api/{acronymCity}
    On the rigth side you can see a list of all available cities. It is important to use the name as it appears.
    You are going to receive a JSON object like:
    {"date": "2015-09-10", "city": "Madrid", "temp": 25}

    Know a specific day weather on a city
    To know the temperature of a city on a specific day it is sufficient to make a request to:

    dailyweather.herokuapp.com/api/{cityName}/{dd}/{mm}/{yyyy}
    or
    dailyweather.herokuapp.com/api/{acronymCity}/{dd}/{mm}/{yyyy}
    You are going to receive a JSON object like:
    {"date": "2015-09-10", "city": "Madrid", "temp": 25}
    Note that if the day that you want is not on you DB you are going to receive a error message
